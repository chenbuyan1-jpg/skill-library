# 本地 Markdown 输出模板

面试复盘文档的本地 Markdown 模板。用于保存到本地文件、Obsidian 或普通 Markdown 阅读器。

核心原则：

- 默认不要使用飞书专用标签。
- 必须保留 [`review-output-framework.md`](review-output-framework.md) 中的整体目录。
- 候选人原回答、Senior 示例、思维路径必须保留代码块。

禁止使用：

```text
<callout ...>
</callout>
background-color="..."
border-color="..."
``` 

---

## 格式规范

| 用途 | 本地 Markdown 写法 |
|------|------------------|
| 面试概览 | 列表，必要时用表格 |
| 岗位画像/口述JD | 标题 + 项目符号 |
| 候选人原回答 | `text` 代码块 |
| 诊断 | `**诊断：**` + 引用块 |
| 关键失分 | `**关键失分：**` + 引用块 |
| Senior 示例 | `text` 代码块 |
| 思维路径 | `text` 代码块，保留 Step 1-4 |
| Level 评级 | 粗体结论 + gap + 补法 |
| 追问防御链 | 列表或表格 |
| 训练任务 | 标题 + 为什么练 + 参考答案 + 来源 + 验收标准 |
| 自查项 | 标题 + 标准答案 + 来源 + 警告 |
| 待确认事项 | 编号列表 |
